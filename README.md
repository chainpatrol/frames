# PhishFrame

![PhishFrame](/public/assets/images/phishframe-cover.png)

PhishFrame is here to give our Farcaster friends a simple and convenient set of security tools to ensure your safety in the world of Web3. Got a URL that's looking a bit fishy? No problem! Just pop it into PhishFrame and with a single click, you can see if it's on ChainPatrolâ€™s scam blocklist.

## Development

First install the dependencies:

```bash
corepack enable
yarn install
```

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
